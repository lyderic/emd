# Fichier de test

Dans ce fichier, nous incorporons quelques particularités de Markdown, afin d'en tester le rendu sur un terminal Linux de base. Par exemple, ces mots devraient apparaître en _italique_ et en **gras**.

To test this file, run :

```bash
$ emd testfile.lkl
```
